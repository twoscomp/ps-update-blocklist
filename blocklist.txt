# =============================================================================
# PlayStation Firmware Update Blocklist for AdGuard Home
# =============================================================================
# Purpose   : Block PS3 / PS4 / PS5 OFW update servers at the DNS level.
#             Prevents automatic and manual firmware updates while still
#             allowing general PSN connectivity (PS Store, trophies, etc.)
# Format    : AdGuard Home / Pi-hole compatible hosts-style blocklist.
#             Each uncommented line is a domain to block.
# Usage     : Add this file as a custom filter list in AdGuard Home under
#             Filters → DNS blocklists → Add blocklist → Add a custom list.
#             Or paste individual entries into Filters → Custom filtering rules.
# Notes     : - PS3 uses only d-prefix (delivery) and f-prefix (fallback) nodes.
#               There is no h-prefix tier on PS3. Additionally, PS3 has a
#               dedicated fshop01 node used when updates are triggered via the
#               PS Store rather than System Settings.
#             - Domain pattern is [region-prefix]01.[platform].update.playstation.net
#               where prefix d = delivery, f = fallback, h = HTTP fallback (PS4/5 only)
#               and region codes: au=Australia, br=Brazil, cn=China, eu=Europe,
#               hk=Hong Kong, jp=Japan, kr=Korea, mx=Mexico, ru=Russia,
#               sa=South Africa, tw=Taiwan, uk=United Kingdom, us=United States
#             - ps5 variants follow the same pattern on .ps5.update.playstation.net
#             - update.playstation.net (bare) and sys.update.playstation.net are
#               also blocked as fallback catch-alls
#             - Sony CDN edge node a01.cdn.update.playstation.org.edgesuite.net
#               and Akamai routing domain playstation.sony.akadns.net are included
#             - This list does NOT block PS Store, PSN auth, or game CDNs.
#               It is scoped to OFW update delivery only.
# Sources   : Al-Azif dns-config-watchdog, depressive_monk hosts list,
#             PSXHAX community blocklist, GBAtemp community research,
#             PS3 Developer Wiki (psdevwiki.com/ps3/Online_Connections)
# Maintained: twoscomp — push updates when new regional nodes are confirmed
# Updated   : 2026-03-11
# =============================================================================
